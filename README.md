## Secure Authentication Lab

An interactive web security simulator built with HTML, CSS, and JavaScript.

## Features
- SQL Injection attack demonstration with real-time mitigation
- Brute-Force attack simulation with account lockout
- Rainbow Table attack with SHA-256 hashing + dynamic salting
- Real-time password strength evaluator
- Admin dashboard with live attack statistics
- CSV audit log generation

## Technologies
HTML5 | CSS3 | JavaScript

## Purpose
Built to demonstrate common web security vulnerabilities and their 
defenses from a developer's perspective.
